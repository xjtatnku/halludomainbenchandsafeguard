# Taxonomy

`scenario_taxonomy.sample.json` defines a seed taxonomy for:

- life domains
- scenarios
- intents
- entry types
- evaluation modes

`prompt_library.starter.v1.json` captures the generated starter prompt variants, open-set prompt topics, and model lineup manifest.

Use `python -m halludomainbench bootstrap-taxonomy` to generate a fresh taxonomy template.
Use `python -m halludomainbench bootstrap-starter-assets` to regenerate the starter prompt library.
